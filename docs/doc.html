"Burnout"

Publisher Author Date Reviewed
WDM Courses Jacob Borg 4/14/2020 NO

Digest

If we are constantly ministering, we will burn ourselves out. Sometimes we need to stop, sit down, and let ourselves be ministered to. We can avoid burnout by setting aside time to be ministered to.
